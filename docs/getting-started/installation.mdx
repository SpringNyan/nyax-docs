# 安装

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

## 安装核心包

使用以下 npm 命令安装 Nyax 的核心包。

```sh
npm i @nyax/core
```

## 安装绑定包

根据选用的状态管理库，安装相应的绑定包及依赖。

## UI 库集成

你可以使用状态管理库官方推荐的方式与 UI 库集成。

MobX 使用 mobx-react-lite 与 React 集成。

```sh
npm i mobx-react-lite
```

Redux 使用 react-redux 与 React 集成。

```sh

```
